## 二 青铜（小白）理解

### 2.第一点：

 > gcc 是编译器，编译器做了什么 就看gcc做了什么
 > 编译器：gcc、llvm 和 clang


源代码 (source code) → 预处理器 (preprocessor) → 编译器 (compiler) → 目标代码 (object code) → 链接器 (Linker) → 可执行程序 (executables)


（1）预处理

   gcc -E hello.c –o hello.i

（2）编译

gcc -S hello.i

hello.s 是一个汇编文件

（3）汇编

 gcc -c hello.s

汇编得到 hello.o 文件

（4）链接

gcc hello.o
